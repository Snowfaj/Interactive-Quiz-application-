## ✨ Features
- 🚀 Start & Retake functionality
- ✅ Score calculation and display
- 📱 Responsive on mobile and desktop
- 🎨 Animated transitions and effects
- 🔁 Randomized or static question sets

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
